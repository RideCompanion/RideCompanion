# RideCompanion
RideCompanion — это открытая система, для поиска попутки и попутчика..

Пользователь - человек, зарегистрированный в системе как водитель или как попутчик.

Попутка — это авто, водитель которого предлагает подвезти попутчика от одной точки в другую.

Попутчик — это человек, который ищет попутку для проезда из одной точки в другую.

Объявление - заполненная форма с данными о точке отправки и точке назначения поездки, ее цене, количестве пассажиров, и данными о пользователе, который создал объявление.

Цена поездки - назначается водителем авто в случае, когда водитель создает объявление о поиске попутчика, либо цена указывает попутчиком, когда водитель ищет объявления попутчика.

Система написана на .NET 7 / PostgreSQL / SQRS, полностью бесплатна и распространяется по свободной, открытой лицензии. Вы в полном праве взять исходный код, и модифицировать его для личных нужд (Не меняя тип лицензии и указывая автора).
